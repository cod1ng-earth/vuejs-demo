## Setup

- Clone repository
  ```
  git clone https://github.com/coding-berlin/vuejs-demo/
  cd vuejs-demo
  ```
  
- Run setup 
  ```bash
  make setup
  ```
    
To verify your installation, open a browser tab:

[http://localhost:8080/](http://localhost:8080/)

## Start existing container

- Run setup 
  ```bash
  make start
  ```
